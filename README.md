## 🚀 Overview

ResumeAI is a modern web application designed to simplify resume creation for students, freshers, and job seekers. It combines traditional resume-building functionality with Generative AI features powered by **Groq API** to help users create stronger and more professional resumes.

---

## ✨ Features

### 📋 Resume Builder
- Create structured and professional resumes.
- Add personal details, education, skills, projects, certifications, and experience.
- Real-time resume preview.
- Clean and responsive user interface.

### 🤖 AI Resume Summary Generator
- Generates ATS-friendly professional career summaries.
- Converts raw input into impactful career descriptions.
- Saves time while improving resume quality.

### 🧠 AI Suggested Skills Generator
- Automatically suggests relevant skills based on user data.
- Instantly adds skills into the resume form.
- Improves job matching and ATS optimization.

### ✉️ AI Cover Letter Generator
- Generates personalized cover letters instantly.
- Tailors content according to job role and resume data.
- Produces professional and structured output.

### 🖨️ Print & Download System
- Print-ready format for Resume & Cover Letter.
- Download complete resume + cover letter package.
- Clean export for job applications.

### 🎬 Embedded Tutorial
- Built-in video tutorial to guide users through the app.
- Helpful for first-time users getting started quickly.

```bash
cd ResumeAI
```

### Run Application

Open `index.html` in your browser, or use VS Code Live Server.

---

## 🔑 Groq API Setup

1. Create an account at [groq.com](https://groq.com).
2. Generate an API key from the Groq Console.
3. Add the API key to your application.

```javascript
const GROQ_API_KEY = "YOUR_API_KEY";
```

> ⚠️ Never commit your API key to version control.
